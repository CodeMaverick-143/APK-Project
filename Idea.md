
# Project Idea Submission Template

1. **Project Title**  
To-Do List with Rewards and Badges

2. **Your Name & Roll Number**  
Arpit Sarang – [2024-B-31082005B]

3. **Problem Statement**  
Many people struggle to stay motivated while managing their daily tasks. Basic to-do lists lack engaging elements to encourage consistent productivity. This project aims to make task management fun and rewarding to boost users’ motivation and help them complete more tasks.

4. **Proposed Solution / Idea**  
A simple, interactive to-do list web app where users can add daily tasks and earn points or badges for completing them. The app will include fun animations and sound effects to celebrate task completion, making the productivity experience enjoyable and rewarding.

5. **Key Features**
- Add, view, and delete daily tasks  
- Mark tasks as completed and earn points  
- Track total points earned by the user  
- Unlock badges when reaching certain point milestones  
- Fun animations and sound effects on task completion  
- Save tasks and points locally so progress is retained across sessions

6. **Target Users / Audience**  
Students, young professionals, or anyone looking for a fun and motivating way to manage daily tasks and improve productivity.

7. **Technology Stack**  
- Frontend: React Native
- Storage: Supabase or Firebase
- Animations: Libraries like animate.css or canvas-confetti  
- Sounds: Howler.js or simple audio files for effects

8. **Expected Outcome**  
A fully functional web app that motivates users to complete daily tasks by rewarding them with points and badges, featuring smooth UI/UX with animations and sound feedback.

9. **Timeline (Optional)**  
- Week 1: UI design, setting up project structure  
- Week 2: Implement task management (add, delete, mark complete)  
- Week 3: Develop reward system (points, badges) and animations  
- Week 4: Add sounds, polish UI, testing, and deployment
